## 🚀 How to Deploy (30 Minutes)

### 1. Create Firebase (5 min)
- Go to console.firebase.google.com
- Create project → Create Realtime Database
- Copy your database URL

### 2. Upload to Vercel (10 min)
- Go to vercel.com
- Import this project
- Set FIREBASE_URL environment variable
- Click Deploy

### 3. Configure Firebase (5 min)
- Add your Vercel domain to authorized domains
- Update security rules
- Test database connection

### 4. Test Everything (10 min)
- Visit your site
- Login to admin
- Test all features
- Verify no errors

**Total Time: 30 minutes**
