## 🚀 3-Step Quick Deploy

### Step 1: Create Firebase (5 min)
1. Go to https://console.firebase.google.com
2. Create new project
3. Create Realtime Database
4. Copy your database URL

### Step 2: Upload to Vercel (15 min)
1. Go to https://vercel.com
2. Import this project (via GitHub or direct upload)
3. Set environment variable: `FIREBASE_URL` = your database URL
4. Click Deploy

### Step 3: Test (10 min)
1. Visit your Vercel URL
2. Login to admin: `admin` / `nextgen2025`
3. Test all features
4. Done! ✅
