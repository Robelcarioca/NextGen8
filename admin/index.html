HOW THIS WORKS — NO DEPLOY BUTTON NEEDED
This system uses Firebase Realtime Database as the single source of truth.
When you save anything in this admin panel, it writes directly to the database.
The live website reads directly from the same database every time a visitor loads a page.
Result: Your changes are live instantly. No export. No upload. No redeploy. Ever.
